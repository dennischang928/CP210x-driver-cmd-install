# CP210x-driver-cmd-install

## Windows
1. Open PowerShell.
2. Run:

   ```powershell
   iwr driver.sengchon.com/install.ps1 | iex
   ```

## macOS
1. Open Terminal.
2. Run:

   ```bash
   curl -fsSL driver.sengchon.com/install-mac.sh | bash
   ```

The macOS installer downloads the official Silicon Labs package from:
https://www.silabs.com/documents/public/software/Mac_OSX_VCP_Driver.zip
